Calculate the slope of a line instantly with our free Slope Calculator. Easily find rise over run, incline angle, distance, and point coordinates. Try it now!

[https://www.calculator.io/slope-calculator/](https://www.calculator.io/slope-calculator/)

== Usage ==

Add the Slope Calculator shortcode to your page, post or sidebar:

`[calcio_slope_calculator]`

Add the following code to your template where you would like the Slope Calculator to appear:

`<?php display_calcio_slope_calculator(); ?>`

1. Upload the Slope Calculator /calcio_slope_calculator/ folder to the /wp-content/plugins/ directory.
2. Activate the Slope Calculator plugin through the "Plugins" menu in WordPress.
